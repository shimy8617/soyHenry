 * Variables
    Una variable es donde se almacenan el valor de algo que vamos a usar más tarde.
    Hay 3 tipos de variables: var, let y const. 

 * Strings
    Es una cadena de texto que se escribe entre comillas. Un bloque de texto.
 * Funciones (argumentos, `return`)
    Una función es un tipo de objeto llamado callable object u objeto invocable.
    a una funcion se le pueden pasar 1 argumento o varios.
    La declaracion del return es la unica forma en que se escapan los datos de la funcion. 
 * Declaraciones `if`
   Es un control de flujo, una forma en que la funcion verifica si algo es true, o si no lo si, sigue avanzando.  
 * Valores booleanos (`true`, `false`)
   Viene de la logica booleana. Significa que hay 2 opciones, 1 o 0, activado o desactivado o en codigo true o false.
   Que significa si algo es verdadero o falso.
